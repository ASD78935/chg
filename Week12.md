#  12주차 TIL: 앱 기본 구조 및 인증/루트 레이아웃

**학습 주제:** 스플래시 화면, 로그인(목업), Root 레이아웃(하단 탭)

## 1. 주요 학습 내용

이번 주차에는 앱의 시작부터 메인 화면 진입까지의 흐름을 구현했습니다.

### A. 스플래시 페이지 (Splash Page)
* **기능:** 앱 실행 시 데이터를 로딩하는 대기 화면 구현.
* **구현:** `SplashController`를 만들어 1.5초 후 자동으로 로그인 화면으로 넘어가게 처리함 (`Future.delayed` 사용).

### B. 인증 프로세스 (Login Page)
* **방식:** Firebase 연동 대신 **UI 중심의 목업(Mock-up)** 방식으로 구현.
* **UI:** 교재 가이드를 참고하여 소셜 로그인 버튼(구글, 애플)이 있는 화면 구성.
* **로직:** 버튼 클릭 시 `Get.offAll()`을 사용하여 뒤로가기가 불가능한 상태로 메인 화면 이동.

### C. 앱 Root 레이아웃
* **하단 탭:** `BottomNavigationBar`를 사용하여 5개의 탭 메뉴 구성.
* **상태 관리:** `GetX`의 `RxInt`로 현재 탭 인덱스를 관리하여, 터치 시 즉시 화면이 전환되도록 구현.

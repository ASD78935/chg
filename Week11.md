# 11주차 TIL: 당근마켓 클론 프로젝트 시작

**학습 주제:** 프로젝트 생성, 라이브러리 설정, 폴더 구조화, 하단 탭 구현

## 1. 주요 학습 내용

이번 주차부터 '당근마켓 클론 코딩' 프로젝트를 시작했습니다. 앱의 기초 공사를 다지는 작업을 수행했습니다.

### A. 프로젝트 환경 설정
* **패키지 설치:** `pubspec.yaml`에 핵심 라이브러리 추가.
    * 상태 관리: `get`
    * UI 유틸: `flutter_svg`, `google_fonts`
* **에셋(Assets) 등록:** 앱에서 사용할 아이콘과 이미지를 `assets/` 폴더에 정리하고 등록함.

### B. MVVM 폴더 구조 설계
* 유지보수를 위해 코드를 기능별로 분리함.
    * `src/pages`: 화면 UI
    * `src/controller`: 비즈니스 로직
    * `src/repository`: 데이터 관리

### C. 하단 내비게이션 (Bottom Navigation)
* **구현:** 5개의 탭(홈, 동네생활, 내 근처, 채팅, 나의 당근)을 가진 하단 메뉴바 생성.
* **상태 관리:** `GetX`를 사용하여 탭을 누를 때마다 화면이 전환되도록 구현.

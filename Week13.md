# 13주차 TIL: 홈 리스트 및 상품 등록

**학습 주제:** 홈 화면 UI, 가짜 데이터 연동, 상품 등록, 바이브 코딩 실습

## 1. 주요 학습 내용

사용자가 가장 많이 쓰는 홈 목록과 글쓰기 기능을 구현하고, 심화 기능을 추가했습니다.

### A. 홈 화면 구성 (Home Screen)
* **UI:** 상단 앱바(지역 선택, 검색)와 `ListView`를 활용한 상품 목록 구현.
* **데이터:** 서버 대신 **Mock Data(가짜 데이터)**를 Repository 패턴으로 연결하여 화면에 바인딩함.
* **기능:** 우측 하단 `FloatingActionButton`을 통해 글쓰기 화면으로 이동.

### B. 상품 등록 페이지 (Write Page)
* **Form:** 제목, 가격, 내용을 입력받는 텍스트 필드 구현.
* **로직:** 입력된 데이터를 컨트롤러로 전달하여, 홈 리스트 최상단에 실시간으로 추가되도록 구현 (`insert` 활용).

### C. 바이브 코딩 & 심화 구현
* **AI 활용:** 반복적인 리스트 아이템 레이아웃이나 데이터 구조를 잡을 때 AI 도구를 활용해 빠르게 작성(Vibe Coding).
* **추가 기능:**
    * **새로고침:** `RefreshIndicator`를 적용해 리스트를 당겨서 갱신하는 기능 추가.
    * **좋아요:** 하트 버튼 클릭 시 즉시 반응하는 상태 관리 로직 구현.
